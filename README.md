# RAG-Based Chatbot using Phi-2

This is a **Retrieval-Augmented Generation (RAG) chatbot** built with **LangChain** and **FAISS**.  
It answers questions **only from the document you provide** and replies **"I don’t know"** for anything outside the document.

---

## Tech Stack
- **LangChain**: RAG pipeline & text splitting  
- **FAISS**: Vector similarity search  
- **Hugging Face Transformers**: Phi-2 model for generation  
- **Sentence Transformers**: For embeddings
